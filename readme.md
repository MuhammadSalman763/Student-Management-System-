# 🎓 Student Management System

A Student Management System developed using **Python**, **FastAPI**, **Pandas**, **Pydantic**, and **Faker**. The project provides RESTful APIs to perform CRUD operations on student records stored in a CSV file.

## 🚀 Features

- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Total Students
- Topper
- Average CGPA
- Department-wise Statistics
- Generate 1000 Fake Students

## 🛠️ Technologies

- Python
- FastAPI
- Pandas
- Pydantic
- Faker
- Uvicorn

## ▶️ Run the Project

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```
